# Time To Travel, 여행 갈 시간!

## 2021-2 지능웹설계 텀프로젝트

지능웹설계 강의 텀프로젝트를 위한 리포지토리입니다.

### 기술 스택

> Java, JSP, Servlet, Tomcat, JDBC

### JVM Arguments

```sh
-Dfile.encoding=UTF-8 -Dusername="DB_USERNAME" -Dcredentials="DB_PASSWORD" -Durl="DB_URL"
```
